I've moved my blog from Wordpress to [Ghost](https://ghost.org/). Because:

1. Very minimalistic.
2. Runs on Node.js, automatic +10 hipster.
3. [Markdown](http://en.wikipedia.org/wiki/Markdown). I **really like** using markdown.
4. This is fun.
